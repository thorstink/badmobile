## Building backend

```bash
mkdir build
cd build
cmake .. 
make
```

## Building UI

We use Elm for now. [Install Elm](https://guide.elm-lang.org/install/elm.html)

## VSCode tasks

Or use the vscode tasks: Ctrl+B for building, and tasks for configuring cmake and running tests. See list of tasks.
